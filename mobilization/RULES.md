# 🎮 Mobilization - Règles du Jeu

## 📋 Objectif
Atteindre la fin du plateau en gérant vos ressources (santé mentale, argent) et en répondant aux exigences des cases.

---

## 🎯 Déroulement d'un Tour

### 1️⃣ **Piocher une carte Événement**
- Piochez **une seule carte Événement** par tour
- Lisez l'événement et appliquez ses effets immédiatement
- Certains événements proposent des choix, d'autres s'appliquent automatiquement

### 2️⃣ **Jouer des cartes Action**
- Vous pouvez jouer **jusqu'à 3 cartes Action** par tour
- Choisissez une carte de votre inventaire qui correspond à l'exigence de la case
- Les cartes ont des coûts monétaires : vérifiez que vous avez assez d'argent

### 3️⃣ **Valider la case**
- Si la carte jouée correspond à l'exigence : la case est validée
- Si la carte ne correspond pas : pénalité appliquée (perte de santé mentale ou d'argent)
- Certaines cartes admin peuvent valider plusieurs types de cases

### 4️⃣ **Passer au tour suivant**
- Cliquez sur "Fin du tour" lorsque vous avez terminé vos actions
- Le tour suivant commence avec réinitialisation des compteurs

## 🏆 Conditions de Fin de Partie

### ✅ **Victoire**
- **Objectif** : Atteindre la dernière case du plateau
- **Résultat** : Diplôme obtenu avec grade selon performance
- **Grades** : A+ (excellent), B (bon), C (passable)

### 💔 **Défaite**
- **Condition** : Santé mentale atteint 0
- **Résultat** : Abandon des études pour épuisement
- **Message** : Contextuel selon la progression accomplie

## 🌍 Deux Modes de Jeu

### 🏠 **Plateau Natif**
- **Personnage** : Étudiant français
- **Contraintes** : Faibles
- **Difficulté** : Modérée
- **Avantages** : Réseau social fort, pas de problèmes administratifs

### ✈️ **Plateau Mobilité**
- **Personnage** : Étudiant international
- **Contraintes** : Élevées (visa, préfecture, logement)
- **Difficulté** : Élevée
- **Avantages** : Expérience internationale, nouvelles perspectives

## 🎲 Stratégies de Base

### 📚 **Optimiser les études**
- Priorisez les cartes "Étudier" pour les cases académiques
- Utilisez les cartes admin pour les problèmes administratifs
- Ne gaspillez pas vos 3 cartes par tour

### 💸 **Gérer l'argent**
- Travaillez régulièrement pour maintenir votre budget
- Anticipez les coûts des cartes "Consulter un médecin"
- Utilisez les événements positifs pour améliorer vos finances

### 🧠 **Préserver la santé mentale**
- Utilisez les cartes "Se reposer" et "Appeler la famille"
- Évitez les événements trop négatifs consécutifs
- Consultez un médecin avant que votre santé ne soit trop basse

---

## ⚠️ Règles Spéciales

### 🚫 **Limitations**
- **1 seul événement** par tour maximum
- **3 cartes Action** maximum par tour
- **Inventaire limité** à 6 cartes
- **Pas de retour en arrière** possible

### 🔄 **Incompatibilités**
- Carte "Étudier" incompatible avec case "Payer le loyer"
- Carte "Travailler" incompatible avec case "Rendre un devoir"
- Certaines cartes admin peuvent valider plusieurs types de cases

### 🎯 **Cas Particuliers**
- **Burn-out** : Si santé mentale = 0, même en cours de partie
- **Deadlock** : Si aucune action possible, système d'aide automatique
- **Événements partagés** : Certains événements affectent tous les joueurs (mode multijoueur)

---

## 🏆 Durée de Partie

- **Objectif** : 45-90 minutes
- **Tours maximum** : Variable selon progression
- **Pause possible** : Entre chaque tour
